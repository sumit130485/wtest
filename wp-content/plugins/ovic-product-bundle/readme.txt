== Description ==

This plugin is supports bundle product WooCommerce.
- simple setting
- usefull
- easy to user

Ovic Product Bundle is a plugin help you bundle a simple products with pre-defined quantity, offer them at a discount and watch the sales go up!
The bundle has smart price calculation so when changing the price of the individual product, the bundle price will be changed too.

== Installation ==

1. Please make sure that you installed WooCommerce
2. Go to plugins in your dashboard and select "Add New"
3. Search for "Ovic Product Bundle", Install & Activate it
4. Now when you create/edit a product you can choose product tab is "Product Bundle", then add many products as you want
5. Done! click Save and see the result

== Installation ==

1. Go to plugins in your dashboard and select "Add New"
3. Search for "Ovic Import Demo", Install & Activate it
4. Now you can find menu Ovic Plugins => Ovic Import Demo and click `Create Import Data` button for create data import in your theme.
